## Description

Starter ASP.NET Core API baked with env variables.

### Tech Stack

- ASP.NET Core Web API
- .NET 6

## Getting Started

### Installation

1. Clone the repo
   ```
   git clone https://github.com/ayush-lal/dotnet6-api
   ```
2. Navigate to project directory and clone .env.example
   ```
   cd src/ExampleAPI
   ```
   - Windows:
     ```
     copy .env.example .env
     ```
   - Linux/MacOS:
     ```
     cp .env.example .env
     ```
3. Run the API
   ```
   dotnet run
   ```
